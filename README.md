<h1 align="center" id="title">DUNE</h1>

<p id="description">This is frontend Dune themed based keyboard project made by nextjs, tailwindcss, gsap and shadcn ui</p>

<h2>🛠️ Installation Steps:</h2>

<p>1. Clone the repository</p>

```
git clone https://github.com/UMDhodi/dunekeyboard_concept.git
```

```
cd dunekeyboard_concept
```

<p>2. Install Dependencies</p>

```
npm install
```

```
npm run dev
```
